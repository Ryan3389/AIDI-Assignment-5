# Failure Cases

## Failure Case 1: Generic and repetitive summary structure

Description:
The generated summaries often follow a very similar structure across different countries.

Example:
Many summaries follow the same pattern:
- country name and region
- capital
- population
- language and currency


## Failure Case 2: Limited usefulness beyond visible data

Description:
Some summaries mainly restate structured data that is already displayed in the UI.

Example:
The summary repeats information like capital, population, and region without adding additional context or insight.
