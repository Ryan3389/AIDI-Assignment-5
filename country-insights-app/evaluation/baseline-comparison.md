# Baseline Comparison

## Baseline System
A simple template-based summary using string formatting without an AI model.

Example:
Canada is in North America. Its capital is Ottawa. Population: 41,651,653.

## Final System
AI-generated summary using the LLM.

## Comparison

Accuracy:
Both systems are accurate because they use the same structured data.

Clarity:
The AI summary is slightly more natural and readable.

Variation:
The template is fully repetitive. The AI shows some variation, but still limited.

Usefulness:
The AI is slightly better, but still mostly restates the same information.

## Takeaway
The AI system provides some improvement in readability and structure, but the gap between the baseline and final system is not very large. This suggests that improving the prompt could make the AI output more valuable.
